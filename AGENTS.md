# Project Instructions

These instructions apply to the entire repository.

- Keep every code file at or below 500 lines.
- Put a short purpose summary comment on the first line of every code file.
- Add a JSDoc/docstring-style summary to every class, function, and method in source code.
- Prefer small, purpose-specific modules over large mixed-responsibility files.
- Run `npm run check` after code changes; it enforces these project guidelines.
